Entendendo o Desafio

Este Desafio de Projeto consiste em criar um website simples, utilizando as principais tags HTML aprendidas nas aulas.

Tags obrigatórias no projeto

São elas:

  1. <h1> até <h6>
  2. <p>
  3. <mark>
  4. <small>
  5. <i>
  6. <u>
  7. <strong>
  8. <ol> e <ul>
  9. <li>
  10. <a>
  11. <hr>
  12. <sub> e <sup>
  13. <blockquote>

Tags sugeridas

  • <font> (obsoleta, mas aceita no desafio)
  • <del>
  • <abbr>

Carreira Dev: Por onde começar?

Dev é uma carreira muito ampla. Para quem está começando, é normal se perguntar: “O que estudar primeiro?”

O primeiro passo é entender as áreas possíveis:

  1. Front-End: Criação da interface que o usuário interage.
  2. Back-End: Lida com regras de negócio, banco de dados e servidores.
  3. Fullstack: Profissional que trabalha tanto no Front quanto no Back-End.

Além disso, é importante desenvolver habilidades fundamentais:

  • Lógica de programação;
  • Versionamento de código com Git;
  • Praticar muito e fazer projetos reais.
"A área de desenvolvimento é muito ampla e constantemente surgem novas tecnologias, por isso é essencial estar sempre se atualizando."

Uma dica importante!

Sempre deixe para depois comece hoje.
A procrastinação é uma das maiores inimigas do seu desenvolvimento profissional.

Busque comunidades, participe de eventos e esteja sempre aprendendo.

Curiosidade

O termo HTML5 significa Linguagem de Marcação de Hipertexto e é a linguagem que estrutura a web.

A tag <sub> é muito usada para representar elementos como fórmulas químicas. Exemplo: H2O (água).
